# HappyHabits Front End

React Native Expo app for HappyHabits.

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npx expo start
   ```

## API Integration

The app is configured to make API calls to the backend using axios.

### Configuration

If running the backend locally, you must change the value of API_BASE_URL in /src/api/config.js to your device's local port 3000.
